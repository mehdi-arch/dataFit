
```
GNU Make 3.81
```
- **C++ Standard:**

```
C++17
```
- **Documentation Generation:**

```
Doxygen 1.10.0
```

## Compilation and Usage

- **Compile the code:**

```
make
```

- **Run the code in /test:**

```
make run
```

- **Generate documentation in HTML and PDF:**

```
make doc
```
